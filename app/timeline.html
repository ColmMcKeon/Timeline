Timeline

What is this?

A visual resource planner — see who is working on what, and when, across a 52-week timeline.

What does it need to run?

Node.js — the JavaScript runtime.
Electron — installed automatically via npm install.
Launch by double-clicking Timeline.app.

How is data saved?

Your data is saved as a JSON file at:
~/Documents/Timeline/timeline-data.json
Press ⌘S or click Save to write to disk. On close, you'll be asked to save if there are unsaved changes.

Keyboard shortcuts

⌘S — Save    ⌘W — Close
